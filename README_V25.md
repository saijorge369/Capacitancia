# V25 — Examen real y modo celular

Cambios:
- Modo examen real: al iniciar, Solución OFF queda bloqueada.
- Durante el examen no se muestra correcto/incorrecto ni clave de la pregunta.
- Al terminar, aparece hoja de resultados con nombre, correo, nivel, clave, intento, nota y retroalimentación.
- Botón Imprimir resultados imprime la hoja final.
- Botón Clave impresa muestra la clave generada.
- Se agregó botón Celular ON/OFF para adaptar la vista a pantallas pequeñas.
- El quiz valida nombre completo y correo antes de iniciar.
